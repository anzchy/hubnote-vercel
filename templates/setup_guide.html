🔑 什么是 GitHub Personal Access Token？

GitHub Personal Access Token 是一种安全的方式来访问 GitHub API，而无需使用您的密码。它就像是一把特殊的钥匙，只允许访问您指定的功能。

为什么需要它？

  • 安全地访问您的 GitHub 仓库和 Issues
  • 不需要存储您的 GitHub 密码
  • 可以随时撤销或更新权限
  • 支持访问私有仓库
2

🛠️ 创建您的 GitHub Token

按照以下步骤在 GitHub 上创建一个新的 Personal Access Token：

  1. 访问 GitHub Token 设置页面
  2. 点击 "Generate new token (classic)"
  3. 为您的 Token 起一个描述性的名称，比如 "HubNote App"
  4. 设置过期时间（建议选择 "30 days" 或 "60 days"）
  5. 选择以下权限（scopes）：
repo 访问所有仓库（包括私有）
public_repo 仅访问公开仓库
read:user 读取用户信息

提示： 如果您只需要访问公开仓库，选择 "public_repo" 即可。如果需要访问私有仓库，请选择 "repo"。
